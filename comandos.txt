git config --global user.name "Nome" – Configura o nome do usuário para todos os commits.
git config --global user.email "email@exemplo.com" – Configura o e-mail do usuário para todos os commits.
git init – Inicializa um repositório Git local.
git status – Exibe o estado atual do repositório, mostrando arquivos modificados e não versionados.
git add <arquivo> – Adiciona o arquivo especificado à área de staging.
git add . – Adiciona todos os arquivos modificados à área de staging.
git commit -m "mensagem" – Cria um commit com as mudanças adicionadas ao staging.
git rm --cached <arquivo> – Remove o arquivo da área de staging, sem apagá-lo do diretório de trabalho.
git remote add origin <URL> – Conecta o repositório local a um repositório remoto.
git push -u origin master – Envia o histórico de commits da branch local para o repositório remoto.
git clone <URL> – Clona um repositório remoto para o repositório local.
git pull – Atualiza o repositório local com as mudanças feitas no repositório remoto.
git checkout <branch> – Altera para a branch especificada.
git merge <branch> – Junta as mudanças da branch especificada com a branch atual.
git branch <nome-da-branch> – Cria uma nova branch.
git rebase <branch> – Reaplica os commits da branch atual sobre a branch especificada.
git revert HEAD – Cria um novo commit que reverte as mudanças do último commit.
git checkout -b <nome-da-branch> – Cria uma nova branch e já altera para ela.
